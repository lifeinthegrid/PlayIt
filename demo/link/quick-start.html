PlayIt - Quick Start

This quick start is designed to get you using PlayIt right away.
For a full rundown on the features and layout of PlayIt see All About PlayIt.

Steps
 

  1. Extract the PlayIt archive to a location on your web server which must be http(s).
  2. Browse to https://your-server/path-to-playit/jquery.playit/demo/quick-start.html
  3. In a text editor open quick-start.xml and begin changing your values
  4. Be sure the cacheXml parameter is set to false until you complete with setup
  5. When moving quick-start.xml and quick-start.html to a new directory be sure to update all relative paths in each file
